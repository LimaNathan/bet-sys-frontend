# Documentação Técnica

Este diretório contém documentação técnica, guias de troubleshooting e notas de desenvolvimento.

## Documentos Disponíveis

### `teste-conectividade.md`
Guia de diagnóstico para problemas de conectividade de rede com Docker.

**Quando usar:**
- Colegas não conseguem acessar sua aplicação na rede local
- Problemas de firewall ou roteamento
- Docker não está acessível externamente

**Conteúdo:**
- Testes de conectividade (ping, telnet, curl)
- Diagnóstico de problemas comuns
- Soluções para bloqueios de rede
